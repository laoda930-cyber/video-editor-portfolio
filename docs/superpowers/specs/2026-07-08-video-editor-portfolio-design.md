# 视频剪辑师个人作品集网站设计说明

## 目标

为视频剪辑师李欣霞搭建一个可运行、可预览的单页个人作品集。网站优先服务 PC 端求职与客户展示，整体暗色、高级、克制，具有明确的影视制作气质，但不使用常见蓝紫霓虹模板风。

## 技术与范围

- React + Vite 单页应用
- 内容版心最大约 1700px，并提供基础平板与移动端适配
- 首版不接后端、CMS、表单提交或真实视频托管
- 作品素材使用抽象动态画面和设计化封面占位，后续可直接替换为真实视频、封面和链接
- 使用原生 CSS 完成视觉与响应式，不引入重型 UI 组件库

## 视觉系统

- 方向：电影片场感与编辑工作台语汇
- 主色：近黑、炭灰、暖白；冷绿色仅用于状态、编号和焦点提示
- 字体：中文无衬线为正文，窄体/等宽风格用于英文标签、时间码与数据
- 细节：宽银幕比例、时间码、细网格、轨道刻度、低对比边框和轻颗粒感
- 动效：缓慢背景位移、卡片悬停和进入过渡；遵守 `prefers-reduced-motion`

## 页面结构

### 1. 全屏 Hero

固定于首屏顶部的导航包含姓名标识、作品、关于、优势和联系锚点。背景使用 CSS 生成的抽象动态影像作为视频占位，并覆盖暗色遮罩、网格与时间码。主标题突出“把节奏、情绪与故事，剪进每一帧”，配视频剪辑师身份、简短能力说明和联系按钮。

### 2. 个人经历

左右非对称布局：一侧为人物剪影式视觉占位，另一侧为个人介绍、电话、邮箱和所在地。以数据块展示半年日均 15 条商业短视频、90+ 企业客户、全流程制作能力，并用时间线呈现实习与两段工作经历。

### 3. 精选作品

使用三张大幅横向作品卡片，分别表达品牌宣传、教育剧情和营销口播。每张卡片含抽象封面、编号、类型、职责标签与播放入口视觉。首版点击仅提供明确的“素材待替换”状态，不伪造真实项目链接。

### 4. 个人优势

四张能力卡片展示高产稳定交付、多行业适配、全流程制作和爆款节奏。卡片采用统一编号、简短说明与编辑轨道式装饰，避免图标堆砌。

### 5. 联系收尾页

整屏深色收尾，使用大标题“下一条好片，从一次对话开始”，突出电话与邮箱，并提供返回顶部操作。联系方式沿用简历内容。

## 组件与数据边界

- `App` 负责页面组合
- `Header`、`Hero`、`About`、`Works`、`Strengths`、`Contact` 各自负责独立区块
- 经历、作品与优势使用本地数据数组渲染，方便后续替换内容
- 不引入全局状态；导航与交互仅使用锚点和局部状态

## 可用性与验证

- 使用语义化区块、连续标题层级、可见键盘焦点和图片替代文本
- 文字与背景保持足够对比度，交互目标不小于 44px
- 检查 1920px、1440px 和 375px 宽度，确保无横向溢出
- 运行构建命令验证 Vite 可正常产出
- 浏览器检查首屏、作品卡片、导航锚点和响应式布局

## 后续扩展

真实素材到位后替换 Hero 视频、人物图、作品封面和作品播放链接；后续参考网站优化仅调整视觉层和内容数据，不改变当前信息架构。
